Ein Repository erstellen

Auf Github ein Repository erstellen (lassen)

In dem Verzeichnis, welches Du mit git verwalten möchtest 


$ git init

Um das Repository dem JugendHackt-github hinzuzufügen 


$ git remote add origin /pfad/zum/repository

Nun nur noch initial comitten & pushen

Ein Repository auschecken


$ git clone /pfad/zum/repository

Oder 


$ git clone benutzername@host:/pfad/zum/repository

Workflow

Geänderte Dateien hinzufügen 


$ git add meine-datei.txt
$ git add .

Commit mit Nachricht abschließen 


$ git commit -m "meine commit Nachricht"
Veränderungen auf den Server laden 


$ git push origin master

WICHTIG!

Um den aktuellen Stand zu bekommen 


$ git pull

Nützlich

Stand des lokalen Repos einsehen 


$ git status
